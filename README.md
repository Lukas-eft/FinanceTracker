# Finance Visualizer

A modern, web-based financial tracking application designed for intuitive expense management and data visualization.

## Features

* Data Visualization: Dynamic donut chart for real-time distribution analysis.
* Expense Management: Interface for adding and tracking individual cost items.
* Financial Metrics: Automatic calculation of total balance, largest expense, and average spending.
* CSV Export: Export financial data for external processing.
* Responsive UI: Clean, card-based design built with Tailwind CSS.

## Tech Stack

* Framework: React 
* Language: TypeScript
* Styling: Tailwind CSS
* Icons: Lucide React 

## Installation

Follow these steps to set up the project locally:

1. Clone the repository:
   ```bash
   git clone [https://github.com/Lukas-eft/finance-visualizer.git](https://github.com/Lukas-eft/finance-visualizer.git)
2. Navigate into the project directory:
   ```bash
   cd finance-visualizer
3. Install all necessary dependencies:
   ```bash
   npm install
4. Start the local development server:
  ```bash
  npm run dev
